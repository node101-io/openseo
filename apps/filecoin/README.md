# OPENSEO - Filecoin Storage Service

This service handles the decentralized storage of Zero-Knowledge Proofs (ZKP) generated by the OpenSEO platform using Filecoin. It ensures that verification records are immutable and permanently accessible.


Decentralized Storage: Stores html files on the IPFS network instead of a centralized server.
Data Integrity: Generates a unique CID (Content Identifier) for every uploaded html.
Immutability: Ensures that verification data cannot be tampered with once stored.


# Installation:
Install the required dependencies:
pnpm install


# Configuration:
Create a .env file in the root directory and add the following variables:
FILECOIN_PORT
---------------------

# Running:
pnpm start
# or from the root directory
pnpm --filter filecoin run start
