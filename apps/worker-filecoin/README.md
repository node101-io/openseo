# OPENSEO - Mock Filecoin Storage Service
This service handles the decentralized storage of Zero-Knowledge Proofs (ZKP) generated by the OpenSEO platform using Filecoin. It ensures that verification records are immutable and permanently accessible.

## Primary Functions
- **Decentralized Storage** Stores html files on the IPFS network instead of a centralized server.
- **Data Integrity** Generates a unique CID (Content Identifier) for every uploaded html.
- **Immutability** Ensures that verification data cannot be tampered with once stored.

## Setup
1. **R2 bucket:** `npx wrangler r2 bucket create openseo-filecoin`  
   Name in `wrangler.toml` must match (`bucket_name = "openseo-filecoin"`).

2. **Install & deploy:**
   ```bash
   pnpm install
   pnpm run deploy
   ```

## Local testing (wrangler dev)

```bash
cd apps/worker-filecoin
pnpm install
pnpm run dev
```

### Env before deploy
- Backend: `FILECOIN_URL=http://localhost:8788`
- Frontend: `NEXT_PUBLIC_FILECOIN_URL=http://localhost:8788`
---

## Env after deploy
- **Backend:** `FILECOIN_URL` = `https://<worker-filecoin>.workers.dev`
- **Frontend:** `NEXT_PUBLIC_FILECOIN_URL` = same URL
